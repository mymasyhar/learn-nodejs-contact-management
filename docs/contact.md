# CONTACT API SPEC

#

## Create Contact API

Endpoint : POST /api/contacts

Headers :

- Authorization : token

Request Body :

```json
{
  "firstName": "Masyhar",
  "lastName": "M",
  "email": "masyhar@example.com",
  "phone": "801234567890",
}
```

Response Body Success :

```json
{
  "data": {
    "id": 1,
    "firstName": "Masyhar",
    "lastName": "M",
    "email": "masyhar@example.com",
    "phone": "801234567890",
  }
}
```
